**geocomp*x*** is a community-driven project supporting **learning and teaching geocomputation** across multiple programming languages.

We create **open educational resources** to promote reproducible geographic analysis in R, Python, Julia, and beyond.

## 🔗 Key Projects

* 📝 [**geocomp*x* blog posts**](https://geocompx.org/post)
* 📘 [**Geocomputation with R**](https://github.com/geocompx/geocompr)
* 🐍 [**Geocomputation with Python**](https://github.com/geocompx/geocompy)
* 🟣 *Geocomputation with Julia* (in development)
* 🗺️ [**Spatial Data Visualization with tmap: A Practical Guide to Thematic Mapping in R**](https://github.com/geocompx/tmap)
* 🐳 [**Docker environments**](https://github.com/geocompx/docker)
